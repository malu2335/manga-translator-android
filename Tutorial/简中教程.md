## 1️⃣ 漫画库页面介绍

<img src="https://raw.githubusercontent.com/jedzqer/manga-translator/main/Tutorial/mangaku-screenshot.jpg" alt="漫画库页面" width="45%" />

漫画库用于 **管理和储存漫画** 📂

* 点击右下角的 **➕ 加号按钮** 可以创建文件夹
* 进入文件夹后，可以添加漫画图片
* ⚠️ **为了保证漫画可以按正确顺序阅读**，请务必确保：

  * 图片的 **文件名顺序** 与漫画的 **阅读顺序一致**，（例如 1.jpg, 2.jpg）

图片添加完成后，就可以前往 **设置页面** 填写 API 信息啦～(ง •̀_•́)ง

## 2️⃣ API 设置页面

本软件需要 **自备 API** 来进行漫画翻译 🛠️

如果你还不了解什么是 API，可以先在网上搜索相关资料，或者**询问AI**，获取一个可用的 API 服务。可以查找一些公益站来获取免费的模型。如果有条件的话，建议使用grok，其对nsfw的容忍度更高。

具体设置步骤如下：

1. **填写 API 地址**

   * 通常是以 `/v1` 结尾的链接

2. **填写 API Key**

   * 🔒 请务必妥善保管
   * **不要泄露给他人**

3. **选择 / 填写模型名称**

   * ⚠️ 模型名称必须 **与供应商提供的完全一致**
   * 建议：

     * 直接从供应商官网复制模型名
     * 或点击 **「获取模型列表」** 按钮进行选择（支持多选，确认后会用英文逗号自动填入）
     *可以重复填入模型以进行比例分流，例如：`A,A,B`（约 2/3 请求用 A，1/3 请求用 B）*

填写完成后就可以开始翻译啦～✨

## 3️⃣ 翻译页面 —— 开始翻译

<img src="https://raw.githubusercontent.com/jedzqer/manga-translator/main/Tutorial/translator-screenshot.jpg" alt="翻译页面" width="45%" />

翻译流程非常简单 💡：

1. 点击 **「从漫画文件夹导入」** 选择漫画目录；如果目录下是多章节子文件夹，会自动创建一个集合并把各子文件夹导入为子章节；如果目录本身直接是图片，则按单本漫画导入
2. 上传完成后，在文件夹内点击 **「翻译文件夹」**
3. 等待翻译完成 ⏳，或在预处理结束后点击 **「开始阅读」**边翻边看

*如果当前漫画是条漫，可以在该漫画文件夹里的 **「阅读设置」** 中把 **「阅读方式」** 切换为 **「条漫滚动」**。*
*首页多选菜单支持一次性批量翻译多个项目。*

*在全文速译开启的情况下，如果页数很多（超过50张），建议先上传一部分图片，再翻译完后再上传一部分，或者划分章节上传。也可以直接去设置里拉高API超时设置和关闭全文速译*

## 4️⃣ 阅读页面

在阅读页面中，你可以进行更精细的调整哦～(๑•̀ㅂ•́)و✧

* 🧩 **拖动翻译好的气泡**，调整到合适的位置

  * 软件会自动保存你的调整

* ❌ 如果 **某一页翻译效果不理想**：

  1. 回到漫画库
  2. 关闭 **「全文速译」** 开关
  3. 长按需要重新翻译的图片
  4. 选中后点击 **「重译」**

* ⚠️ 如果 **大量页面翻译效果不佳**：

  * 关闭 **「全文速译」** 后全部重译
  * 还是不行就删除漫画重新上传，并切换到 **更智能 / 更高级的AI模型** 🤖✨

* 💬如果 **一些文字区域没有识别**：

  1. 点击阅读页面右上角的编辑按钮
  2. 点击加号按钮新建一个空白气泡
  3. 长按气泡调整到合适大小，并拖动到未识别文字区域
  4. 点击右上角的绿勾退出编辑
  5. 等待该区域重新识别并翻译

* 🎚️ 如果你觉得气泡太挡画面：

  * 可以在设置里调整 **游离气泡透明度**

* 双击可以进入放大状态，再次双击退出

## 5️⃣ 悬浮窗翻译

使用方式：

1. 在漫画库页面点击 **「悬浮窗翻译」**
2. 按提示授予 **悬浮窗权限** 和 **屏幕捕获权限**
3. 回到桌面或任意 App 后，右侧会出现一个 **悬浮球**

悬浮球交互逻辑：

* **单击悬浮球**：立即识别并翻译当前屏幕中的文本区域
* **双击悬浮球**：清除当前屏幕上已经显示的识别/翻译气泡
* **长按悬浮球**：弹出菜单
  * **编辑模式**：进入后悬浮层会拦截触摸，你可以长按气泡来删除已有气泡，或点击 **「新增气泡」** 后拖动手指框选矩形区域来手动补一个空白气泡，还可以拖动气泡来移动气泡位置
    * **确认**：只对当新增空白气泡补做识别与翻译，不会重跑已有译文
    * **取消**：放弃本次编辑模式里的新增、删除和拖动修改
  * **退出**：关闭悬浮窗翻译
  * **划屏翻译**：启动后不识别气泡，直接进入编辑模式，由用户手动框选区域
悬浮窗编辑模式推荐流程：

1. 先 **单击悬浮球**，让软件识别当前屏幕已有文本区域
2. 如果有翻译效果极差导致无法阅读，**长按悬浮球** 打开菜单，点击 **「编辑模式」** 调整翻译气泡

补充说明：

* 悬浮窗编辑结果只保留在当前会话中；再次单击识别、双击清空或退出悬浮窗后会重置
* 如果提示当前功能需要 VL 模型，说明你现在填写的悬浮窗模型不支持图片输入，请更换模型或关闭该开关
* 如果点击悬浮球后没有识别结果，先检查屏幕捕获授权和 API 配置是否正常

## 6️⃣ 其它设置

### OCR 设置（API OCR）

在设置页点击 **「OCR 设置」** 可以切换 OCR 模式：

* 开启 **「使用本地 OCR」**：使用软件内置 OCR（离线）。
* 关闭 **「使用本地 OCR」**：使用 API OCR（在线），需要填写 OCR API 地址、Key、模型名称。
* 使用 API OCR 做 **多语言翻译** 时，不受“日文/英文”选项限制；只要 **API OCR 模型** 和 **翻译模型** 支持该语言即可。
* 如果 OCR 经常超时，可以在 OCR 设置里提高 **「OCR API 超时（秒）」**（范围 30–1200）。
* 云端 OCR 不消耗本地算力，如果配置较差推荐启用，如果想获得更好的识别效果也推荐使用，但不建议小白使用，因为对模型要求高。

### 悬浮窗翻译设置

在设置页点击 **「悬浮窗翻译设置」** 可以单独配置悬浮窗翻译使用的 API：

* 这里填写的 **API 地址 / Key / 模型名称** 只作用于悬浮窗翻译。
* 如果某一项留空，会自动回退到主设置里的对应配置。
* 悬浮窗里的 **模型名称** 也支持用英文逗号分隔多个模型来分流，例如：`A,B,C`
* 可以开启 **「使用 VL 模型直接翻译整个气泡」**：
  * 开启后会把检测到的气泡截图直接交给视觉模型翻译，省去 OCR 步骤
  * 需要使用支持图像输入的 **VL / 多模态模型**
* 可以设置 **「VL 翻译线程数」**
  * 只在开启 **VL 直翻** 时生效
  * 线程数越高，同时翻译的气泡越多，速度通常更快，但也会更占用 API 并发和额度
* 如果开启这个开关后调用失败，通常是因为当前模型**不支持图片输入**，这时请改成 VL 模型，或者关闭该开关。
* 启用校对模式会在每次识别后自动进入编辑模式，交给用户自行调整
* 启用自动清除气泡，可以在翻页时自动关闭翻译气泡，但会增加一些性能消耗

## 软件使用小技巧

- **跳转页面**：当你想要跳转到某一页面进行观看时，在漫画库的文件夹里面，点击对应的图片文件名称即可完成跳转。
- **编辑气泡**：点击阅读页面右上角的编辑按钮进入编辑模式，长按气泡可以修改气泡大小或删除气泡，点击右上角的加号可以新建气泡；如果自动检测没覆盖到文字，也可以手动补一个空白气泡或游离气泡。
- **导出漫画**：如果你不喜欢在软件里面阅读，可以在文件夹内使用导出功能。导出目录通常为你选择的文档目录下的 `manga-translate` 文件夹；未额外选择时，默认提示路径一般是 `/Documents/manga-translate`。
- **漫画文件夹导入**：在漫画库页点击 **「从漫画文件夹导入」** 后，软件会先检查你选中的目录结构：
  1. 如果该目录下存在多个 **包含图片的子文件夹**，软件会把这个目录当作一部漫画的上级目录处理：先让你填写集合名称，然后自动创建一个 **文件夹集合**，再把这些子文件夹按原名称一起导入为 **子章节**。
  2. 如果该目录本身直接包含漫画图片（jpg/jpeg/png/webp，Android 14+ 额外支持 avif），软件会把它当作 **单本漫画文件夹** 处理，并弹出命名窗口，导入为普通文件夹。
  3. 如果目录下既没有可导入的图片，也没有包含图片的子文件夹，就会提示没有可导入内容。
- **集合内导入子章节**：进入某个 **文件夹集合** 后，点击 **「导入子章节」**：
  1. 如果你选中的目录下有多个 **包含图片的子文件夹**，软件会弹出多选列表，让你勾选要导入的子文件夹，并按原文件夹名导入到当前集合下。
  2. 如果你选中的目录本身直接是图片，软件会把当前目录当成一个子章节导入到该集合中，章节名使用当前目录名。
  3. 已存在同名子章节、空文件夹、没有图片的目录会被自动跳过，并在导入完成后显示导入数量与跳过数量。
- **CBZ/ZIP 导入**：在漫画库页点击 **「从 CBZ/ZIP 导入」** 可直接导入压缩包，压缩包内需为图片文件（jpg/jpeg/png/webp，Android 14+ 额外支持 avif）。
